## 🌟 Overview

A premium, high-performance portfolio designed to showcase technical expertise in software engineering and security. This project leverages the latest features of **Next.js 16**, **TypeScript**, and **Tailwind CSS** to deliver a cinematic user experience with a "Dark-First" aesthetic.

### ✨ Key Highlights
- **Real-time GitHub Integration**: Dynamic fetching of repository data and stars via the GitHub API.
- **Cinematic Animations**: Sophisticated scroll-triggered transitions and interactive elements powered by Framer Motion.
- **Security-Centric**: Specifically highlights cybersecurity research and penetration testing capabilities.
- **Optimized Performance**: Built with App Router for lightning-fast routing and server-side optimization.

## 🚀 Installation & Local Setup

Get the development environment running on your machine:

### 1. Clone & Navigate
```bash
git clone https://github.com/Denis-7242/portfolio.git
cd portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the site.
