## 🛠️ Installation & Setup

Get a local copy up and running in minutes:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Denis-7242/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
4. **Open the browser**
   Visit `http://localhost:3000` to see the site in action.
